# Roadmap

## Phase 1: Portfolio-ready

- Static dashboard
- Domain layer
- Mock API
- Tests
- GitHub Actions
- GitHub Pages workflow

## Phase 2: Real backend

- NestJS или FastAPI API
- PostgreSQL schema
- Prisma/TypeORM или SQLAlchemy
- JWT auth
- RBAC middleware
- Swagger/OpenAPI

## Phase 3: Production behavior

- Redis cache для KPI
- Queue для уведомлений
- Audit log table
- Integration retry policy
- Structured logs
- Monitoring dashboard

## Phase 4: Frontend scale

- React/Next.js migration
- Component library
- Route-level views
- Optimistic UI updates
- E2E smoke tests
